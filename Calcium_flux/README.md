# CaOs (Calcium Oscillation) Analysis Pipeline

Analysis pipeline for evaluating fluorescence intensity data, specifically focusing on calcium oscillations in experiments involving synthetic immunological synapses.

## Project Structure
- `utils.py`: Utility functions for data processing and plotting.
- `cell_detection_CaFlux.py`: Main script to detect cells in the fluorescence images, track them, and measure their intensity. 
- `analysis_script.py`: Main script that executes the trace processing and classification.
